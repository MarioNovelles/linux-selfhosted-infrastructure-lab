## Scripts

This repository includes small Bash scripts used for maintenance and lab administration tasks.

| Script | Purpose |
|---|---|
| [`update-laptop.sh`](./update-laptop.sh) | Updates APT packages, Flatpaks, selected Docker services, Neovim, LazyVim, and related local tooling |
| [`update-ubuntu-server.sh`](./update-ubuntu-server.sh) | Updates Ubuntu Server packages and performs basic package cleanup |
| [`start-jellyfin.sh`](./start-jellyfin.sh) | Starts the local Jellyfin-related workflow used in the lab |
