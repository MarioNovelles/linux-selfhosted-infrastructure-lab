# Examples

This folder is reserved for sanitized example configurations from the lab.

The examples are intended to show structure and operational thinking without publishing real production configuration.

## Planned Examples

- `searxng/` — sanitized Docker Compose example for SearXNG with JSON search support
- `uptime-kuma/` — sanitized Docker Compose example for monitoring
- `monitoring/` — sanitized Prometheus and Grafana example
- `media-service/` — sanitized example for a media-related service

## Safety Notes

Examples in this folder should not include:

- real domains
- real public IP addresses
- internal IP addresses
- passwords
- API keys
- tokens
- private volume paths
- real `.env` files
- production database credentials
- production firewall or reverse proxy configuration
